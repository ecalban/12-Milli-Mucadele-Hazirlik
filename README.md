# Millî Mücadele Hazırlık Dönemi · Eşini Bul

RTF notlarındaki 152 bilgiyi tanım ve cevap kartlarıyla eşleştiren, bağımlılıksız ve mobil uyumlu web oyunu.

Eşleşen kartlar aktif desteden çıkarılır; aynı kart ancak deste baştan karıştırılırsa tekrar görünür.

## Canlı bağlantı

https://ecalban.github.io/12-Milli-Mucadele-Hazirlik/

## Çalıştırma

`index.html` dosyasını doğrudan tarayıcıda açabilirsiniz.

İsterseniz klasörde basit bir yerel sunucu da başlatabilirsiniz:

```bash
python3 -m http.server 8000
```

Ardından `http://localhost:8000` adresini açın.
