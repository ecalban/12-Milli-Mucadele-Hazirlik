# Milli Mücadele Hazırlık Kartları

Bu statik uygulama, verilen RTF dosyalarındaki bilgileri iki bağımsız alıştırmaya dönüştürür.

`index.html` dosyasını tarayıcıda açarak çalıştırabilirsiniz. İlk bölüm kategori kartlarıyla, ikinci bölüm tanım-cevap eşleştirmeleriyle çalışır. Kontrol ekranları cevap anahtarı vermez; yanlış kartlar yalnızca işaretlenir ve ipucu paneli sadece seçili karta ait ipucunu gösterir.
